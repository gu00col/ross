Como Funciona

Três passos simples para análises profissionais.

1
Upload do Contrato

Faça upload do seu contrato em PDF, Word ou texto. Sistema seguro e dados não armazenados.

2
Análise Automática

IA especializada analisa cada cláusula, identifica riscos e mapeia oportunidades de melhoria.

3
Relatório + Chat

Receba relatório completo e converse com a IA para esclarecer dúvidas e refinar estratégias.

Política de Privacidade
Privacidade Total
Ross é self-hosted. Seus dados permanecem em sua infraestrutura.
1. Sobre o Ross

O Ross é um software open-source para análise jurídica de contratos com IA, disponibilizado sob licença MIT. O software funciona de duas formas:

  • Self-hosted: Instalação em seus próprios servidores
  • Demo online: Versão de teste hospedada em ross.dev.br
2. Dados na Versão Self-hosted

Na versão self-hosted (instalação própria):

  • Não coletamos dados: Todos os dados permanecem em sua infraestrutura
  • Controle total: Você possui controle absoluto sobre todos os dados processados
  • Sem telemetria: O software não envia informações para servidores externos
  • Responsabilidade: A proteção dos dados é responsabilidade do administrador da instalação
3. Dados na Demo Online (ross.dev.br)

Na versão demo hospedada:

  • Armazenamento temporário: Contratos enviados são armazenados por no máximo 24 horas
  • Exclusão automática: Após 24h, todos os dados são permanentemente removidos
  • Finalidade: Demo destina-se apenas a testes e demonstração das funcionalidades
  • Não usar dados reais: Recomendamos não enviar contratos com dados sensíveis na demo
4. Dados de Contato

Quando você entra em contato conosco:

  • Formulários: Nome, e-mail e mensagem são utilizados apenas para responder sua solicitação
  • WhatsApp: Conversas via WhatsApp seguem a política de privacidade do WhatsApp
  • Não compartilhamos: Seus dados de contato não são compartilhados com terceiros
5. Cookies e Rastreamento
  • Site institucional: Utilizamos apenas cookies essenciais para funcionamento
  • Sem analytics intrusivos: Não utilizamos Google Analytics ou ferramentas similares
  • Self-hosted: Na sua instalação, você controla completamente cookies e logs
6. Seus Direitos

Você tem direito a:

  • Solicitar informações sobre dados que possamos ter
  • Solicitar correção de dados incorretos
  • Solicitar exclusão de dados de contato
  • Instalar e usar a versão self-hosted sem qualquer coleta de dados
7. Contato
